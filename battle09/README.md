# BattleMap 作战地图 v0.9.0

> 整合版：统帅驾驶舱 + 战区指挥台 + PRDbygemini 功能

## 简介

BattleMap 是一个面向金融租赁行业的产业客户作战指挥平台，定位为「GIS（地理信息）+ 产业大数据 + 协同指挥」的投行级资产管控与创利沙盘。

本版本（v0.9.0）整合了 PRD.md 的精细化数据管理能力和 PRDbygemini.md 的战略指挥视角，新增统帅驾驶舱和战区指挥台。

## 核心功能

### 统帅驾驶舱（新增）

- **北极星 KPI 看板**：实际投放额、融资租赁余额、渗透率、IOT在线率、逾期敞口
- **红绿灯资产预警网**：健康/关注/高危节点可视化，闪烁高亮高危企业
- **区域视角热力图**：按投放/渗透率/不良率着色
- **防雷预警监控**：逾期+IOT离线+中登抵押+信用评分下降联合预警

### 战区指挥台（新增）

- **框选派单**：地图绘制多边形，批量派发工单
- **鹰眼轨迹追踪**：客户经理拜访路线图 + 打卡节点
- **网格渗透率监控**：园区渗透率达标热力

### 原有功能

- 宏观态势面板（全国热力地图 + 园区散点）
- 产业链沙盘（树形图 + 力导向图谱 + 融资结构）
- 客户全景视图（负债旭日图 + 六维雷达 + IoT时序）
- 个人工作台（工单管理 + 商机发现 + 数据补录）
- 规则配置（数据目录 + 业务规则画布）

## 快速开始

```bash
cd battle09/frontend
npm install
npm run dev
```

访问 http://localhost:5173

## 技术栈

- React 18 + TypeScript 5
- Vite 8
- Zustand 5（状态管理）
- Tailwind CSS 4 + CSS 变量（双主题）
- ECharts 5（图表 + 地图）

## 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 宏观态势 | 全国产业地图 + 指标展示 |
| `/chain` | 产业链沙盘 | 产业链拓扑 + 融资结构 |
| `/customer` | 客户全景 | 企业详情 + 触达路径 |
| `/tasks` | 个人工作台 | 工单闭环 + 商机发现 |
| `/commander` | 统帅驾驶舱 | KPI看板 + 红绿灯预警 |
| `/battlefield` | 战区指挥台 | 框选派单 + 鹰眼轨迹 |
| `/rules` | 规则配置 | 数据目录 + 策略画布 |

## 数据管理

系统支持 18 种数据类型的 CSV/JSON 导入，导入后所有视图实时联动。

入口：`个人工作台` → `数据补录`
