# 作战地图 — 产业金融可视化作战系统

> 版本：v2.0.1 | 更新日期：2026-04-14 | 状态：Demo 完善版

产业金融领域的可视化作战平台，聚焦宏观态势感知、产业链分析、客户全景洞察、个人工单管理、数据规则配置五大核心模块，帮助客户经理和业务团队快速掌握战场全局。

---

## 快速启动

### 环境要求

| 工具 | 版本 |
|------|------|
| Node.js | 18+ |
| npm | 9+ |

### 启动步骤

```bash
# 进入前端工程目录
cd battlemap05/frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev

# 浏览器打开
http://localhost:5173/
```

### 生产构建

```bash
npm run build      # 产物输出到 dist/
npm run preview    # 本地预览生产构建
```

## 六大功能模块

### 模块一：宏观态势感知 (`/`)

三种地图视图自由切换，每种视图左侧面板联动更新：

| 视图 | 地图渲染 | 左侧面板 |
|------|---------|---------|
| 行政区 | 省级热力图（Choropleth），颜色深浅映射选中指标 | 省份选择器 + 指标排行 |
| 产业园区 | 散点气泡图，大小映射指标值，金融机构飞线 | 园区选择器 + 客户经理展开 |
| 人员 | 按客户经理分色标注覆盖范围 + 拜访路径连线 | 团队成员列表，点击聚焦 |

**顶部 11 个 KPI 卡片** 可点击切换地图着色指标（投放 / 融资租赁 / 企业贷款 / 全量融资 / 活力指数 / 开工率 / 金租渗透率 / 用电增速 / 设备投放 / 客户数 / 不良率）。

支持**分屏对比**：左右双地图 + 底部 11 维并列柱状图。

---

### 模块二：产业链结构化沙盘 (`/chain`)

**双视图切换：**
- **树形图**：自上而下层级布局，节点内显示环节名称 + 规模 + 景气度
- **力导向图谱**：圆形节点 + 箭头连线，可拖拽

**右侧联动面板**：点击任意节点 → 实时刷新开工景气度 / 银行占比 / 金租占比 + 融资结构条 + 链上企业列表（可跳转客户全景）。

---

### 模块三：客户全景视图 (`/customer`)

页面分为三大区块：

**区块1 — 顶部固定栏**：企业头像 + 名称 + 机器人 AI 问答入口 + 触达路径按钮 + 建立工单卡片 + 管户客户经理卡片。点击触达路径弹出全屏链路图（存量朋友圈 / 产业链中转 / 协会机会）；点击建立工单弹出营销方案弹窗。

**区块2 — 三列布局：**
- 负债结构剖析（旭日图 + 短贷错配风险摘要 + 到期时间表）
- 动态基准线对标（六维雷达图：企业 / 园区均值 / 全国均值）
- 指标监测（双 Y 轴折线图，下拉切换：IoT 设备监测 / 数智前瞻指标 / 基准线指标）

**区块3 — 底部客户资信报告**：综合评分环 + 6 项关键指标 + 风险提示，宽度与区块2对齐。

---

### 模块四：个人工作台 (`/tasks`)

**主 Tab 导航**：我的待办 / 全部任务 / 我的已办 / 我的分配，支持按红/黄/绿分类过滤。

**右侧面板**（点击工单后展示）：跟进详情时间线，支持添加跟进记录。

**辅助功能入口**：任务分发 / 商机发现 / 客户检索 / 数据补录，点击后在右侧面板展开对应功能界面。

**内联 AI 助手**：工作台标题旁圆形机器人图标，点击弹出 AI 对话窗口。

---

### 模块五：规则配置 (`/rules`)

| Tab | 功能 |
|-----|------|
| 数据目录库 | 全域数据字段映射表（原始字段 → 业务名称 → 格式 → 来源） |
| 业务规则画布 | 规则列表 + 可视化条件编排（变量 + 运算符 + AND/OR） + 触发动作 + 统计 |
| 策略沙盒回测 | 选择规则 + 时间区间 → 运行回测 → 输出触发频次/召回率/误报率报告 |

---

### 模块六：数据补录 (`/supplement`)

**核心理念：** 按对象类型统一导入导出，支持模板下载，数据补录后全系统联动更新。

**布局**：左侧类型选择器 + 右侧详情面板

**左侧类型选择器**：
- 6 个分组（行政区划 / 行业与产业 / 核心实体 / 关系数据 / 事件记录 / 指标数据）
- 每个类型显示：标签 / 当前条数 / 来源标签（已导入/示例数据）
- 搜索过滤 / 分组折叠展开

**右侧详情面板**：
- 顶部：类型名称 / 分组 / 来源 / 条数 / 最近导入时间
- CSV 必需列说明
- 文件上传区（拖放 + 点击上传，支持 .json 和 .csv）
- 操作按钮：JSON 模板下载 / CSV 模板下载 / 导出当前数据 / 恢复示例数据
- 底部：变更记录（该类型的最近导入日志）

**全量导出**：左侧底部「导出全量数据」按钮，下载包含所有类型的 JSON 文件

**数据联动**：导入数据后，`宏观态势` / `产业链沙盘` / `客户全景` / `个人工作台` 所有视图实时同步更新

## 主题切换

系统支持**深色模式**和**浅色模式**实时切换，入口在侧边栏底部按钮。

切换机制：`data-theme="dark|light"` + 30+ CSS 变量 + `useChartTheme()` hook，所有 ECharts 图表均响应主题切换，0.25s 平滑过渡动画。

## 后端接入指引

将 Mock 数据替换为真实 API 时，按以下结构接入：

```
GET /api/macro/admin?metric=投放           → 行政区热力数据
GET /api/macro/parks?metric=投放           → 产业园区散点数据
GET /api/chain/topology?industry=新能源    → 产业链拓扑数据
GET /api/customer/:id                      → 企业全息档案
GET /api/tasks?status=all                  → 工单列表
GET /api/tasks/:id/follow                  → 跟进记录
GET /api/rules                             → 规则列表
GET /api/rules/:id/backtest                → 回测结果
```

修改 `src/services/apiService.ts` 中的 `DATA_SOURCE` 开关即可切换数据源。
